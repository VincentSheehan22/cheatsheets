# Markdown Cheatsheet

## Headings
```
# Heading 1
## Heading 2
### Heading 3
```

> # Heading 1
> ## Heading 2
> ### Heading 3

## Unordered Lists
```
* Item 1
* Item 2
    * Sub-Item 1
- Item 3
- Item 4
    - Sub-Item 1
```

> * Item 1
> * Item 2
>     * Sub-Item 1
> - Item 3
> - Item 4
>     - Sub-Item 1

## Ordered Lists
```
1. Item 1
2. Item 2
    2.1 Sub-Item 1
3. Item 3
4. Item 4
    4.1 Sub-Item 1
```

> 1. Item 1
> 2. Item 2
>     2.1 Sub-Item 1
> 3. Item 3
> 4. Item 4
>     4.1 Sub-Item 1

## Tables
```
Heading 1    | Heading 2       | Heading 3
:---         | :---:           | ---:
left-aligned | centre-aligned  | right-aligned
1            | 1               | 1
2            | 2               | 2
3            | 3               | 3
```

> Heading 1    | Heading 2       | Heading 3
> :---         | :---:           | ---:
> left-aligned | centre-aligned  | right-aligned
> 1            | 1               | 1
> 2            | 2               | 2
> 3            | 3               | 3

## Checkboxes
```
* [ ] Unticked
* [x] Ticked
```

> * [ ] Unticked
> * [x] Ticked

## Code Blocks
### Fenced
    ```
    code
    ```

> code

### Tabs
```
    code
```

> code
