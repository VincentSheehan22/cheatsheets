# Markdown Cheatsheet

## Headings
```
# Heading 1
## Heading 2
### Heading 3
```

> # Heading 1
> ## Heading 2
> ### Heading 3

## Tables
```
Heading 1    | Heading 2       | Heading 3
:---         | :---:           | ---:
left-aligned | centre-aligned  | right-aligned
1            | 1               | 1
2            | 2               | 2
3            | 3               | 3
```

> Heading 1    | Heading 2       | Heading 3
> :---         | :---:           | ---:
> left-aligned | centre-aligned  | right-aligned
> 1            | 1               | 1
> 2            | 2               | 2
> 3            | 3               | 3

## Checkboxes
```
* [ ] Unticked
* [x] Ticked
```

> * [ ] Unticked
> * [x] Ticked

## Code Blocks
### Fenced
    ```
    code
    ```

> code

### Tabs
```
    code
```

> code
